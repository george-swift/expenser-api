## Expense Tracker Backend

This repository contains the backend implementation for the Expense Tracker app. The backend is built using Python and deployed on AWS with a serverless architecture. It provides RESTful APIs for managing user expenses, categories, and merchants.

### Features

- Expense Management: Create, update, retrieve, and delete expense records.
- Category Service: Manages and retrieves categories for expenses.
- Merchant Service: Provides merchant suggestions for expense entries.
- Serverless Deployment: Powered by AWS Lambda, API Gateway, and DynamoDB for scalability and cost-efficiency.

### Tech Stack

- Language: Python (3.12+)
- Framework: FastAPI / AWS Lambda Handlers
- Database: DynamoDB
- Authentication: Auth0 (planned)
- Deployment: AWS
